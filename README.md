# Data Extraction & Excel Exporter

A clean Python-based data parsing tool designed to extract, clean, and export structured data into formatted Excel (.xlsx) spreadsheets.

## Features
- **Structured Data Modeling:** Converts raw key-value objects into Pandas DataFrames.
- **Automated Excel Export:** Generates clean .xlsx files using openpyxl.
- **Error Handling & File System Verification:** Validates path outputs dynamically.

## Tech Stack
- **Language:** Python 3.8+
- **Data Manipulation:** Pandas
- **Excel Engine:** OpenPyXL

## Quick Start
1. Install dependencies:
   pip install pandas openpyxl

2. Run the script:
   python scraper.py
